Converte dados de dieta para um arquivo binario.

Esta funcao recebe um array de estruturas Dieta e o numero total de elementos (tamanho). Em seguida, ela cria um arquivo binario e escreve os dados da estrutura Dieta no formato binario. O arquivo resultante e util para armazenar dados de dieta de forma compacta e eficiente.

Parâmetros
nomeFicheiroBinarioNome do arquivo binario a ser criado.
dietaArray de estruturas Dieta contendo os dados a serem convertidos.
tamanhoNumero total de elementos no array.
Retorna
true se a conversao foi bem-sucedida, false em caso de erro.

Paciente  pacientes[],
int  maximoPacientes 
)

Importa informacoes de pacientes a partir de um arquivo.

Esta funcao le informacoes de pacientes de um arquivo CSV e popula um array de pacientes com os dados lidos.

Parâmetros
nomeFicheiroO nome do arquivo CSV contendo as informacoes dos pacientes.
pacientesUm array de pacientes a ser preenchido com as informacoes do arquivo.
maximoPacientesO numero maximo de pacientes que podem ser lidos do arquivo.
Retorna
Retorna o numero de pacientes importados com sucesso. Retorna 0 se houver um erro ao abrir o arquivo ou se nao houver pacientes a serem lidos.
